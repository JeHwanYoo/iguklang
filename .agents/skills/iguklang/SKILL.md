# iguklang

이국랭은 [Brainfuck](https://esolangs.org/wiki/Brainfuck)과 명령이 1:1 대응하는 난해한 프로그래밍 언어다. 8비트 셀 32,768개로 이루어진 배열 위에서 포인터를 움직이며 값을 조작한다. 실행기는 npm 패키지 `iguklang`으로 제공된다.

## 실행 방법

```sh
npx iguk <파일.iguk>
# 또는 전역 설치 후
npm install -g iguklang && iguk <파일.iguk>
```

## 명령어 (Brainfuck 대응)

| 이국랭 | Brainfuck | 동작 |
|--------|-----------|------|
| `이구`(`우`×n)`욱` | `+`×n | 현재 셀 값을 `우` 개수만큼 증가 |
| `이구`(`구`×n)`국` | `-`×n | 현재 셀 값을 `구` 개수만큼 감소 |
| `고수?` | `>` | 포인터 +1 |
| `하-` | `<` | 포인터 −1 |
| `신` | `[` | 현재 셀이 0이면 대응 `킹갓 충무공 제너럴` 다음으로 점프 |
| `킹갓 충무공 제너럴` | `]` | 현재 셀이 0이 아니면 대응 `신`으로 점프 |
| `이국 왤캐 고수임?` | `,` | 입력 한 글자의 ASCII 코드를 현재 셀에 저장 |
| `이국이 처럼 살고싶다.` | `.` | 현재 셀 값을 ASCII 문자로 출력 |

- 주석은 `//` 로 시작해 줄 끝까지.
- 셀 값은 256으로 wrap-around(8비트). 포인터는 0부터 시작.
- `이구`와 종결(`욱`/`국`) 사이의 `우`/`구` 개수가 증감량이다. 예: `이구우우우욱` = +3, `이구구구국` = −2.

## 작성 절차

1. Brainfuck으로 먼저 알고리즘을 설계한다(메모리 레이아웃, 셀별 역할).
2. 각 Brainfuck 명령을 위 표대로 이국랭 키워드로 치환한다.
3. 증감은 반복 대신 `우`/`구` 개수로 한 토큰에 몰아 쓰는 편이 짧다.
4. `.iguk` 파일로 저장하고 `npx iguk`로 실행해 출력을 확인한다.

## 예제: "Hi" 출력

```
// 셀0을 'H'(72)로: 이구 + 우×72 + 욱
이구우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우욱
이국이 처럼 살고싶다.
// 'i'(105) = 'H'(72) + 33
이구우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우우욱
이국이 처럼 살고싶다.
```

셀 값이 크면 `신`/`킹갓 충무공 제너럴` 루프로 곱셈을 써서 토큰 수를 줄이는 것이 관용적이다(예: 셀A에 10을 넣고 루프에서 셀B를 7씩 더해 70을 만든다). `README.md`의 Hello, World! 예제가 대표적이다.

## 주의

- 무한 루프에 취약하다. 루프의 카운터 셀이 반드시 0에 도달하는지 확인한다.
- 입력(`이국 왤캐 고수임?`)은 stdin에서 1바이트씩 읽는다. 대화형 입력은 물론 파이프·리다이렉트(`echo AB | iguk prog.iguk`)도 순차로 처리되고, 개행(ASCII 10)도 입력 가능하다. EOF에 도달하면 셀에 0이 들어간다.
